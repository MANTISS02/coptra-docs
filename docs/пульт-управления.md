# Пульт управления

Универсальный пульт управления для всех типов дронов.

## Характеристики

- Дальность связи: до 10 км
- Время работы: до 8 часов
- Встроенный дисплей: 5.5 дюймов
- Поддержка телеметрии

## Функции

- Настраиваемые элементы управления
- Поддержка автоматических режимов полета
- Запись полетных данных
- Интеграция с мобильными устройствами
